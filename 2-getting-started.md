## 2. Getting Started

Before you can start working, you need to set up Git on your local machine and grab the code.

### Step 1: Install Git
**For Mac (using Homebrew):**
```bash
brew install git
```
**For Ubuntu/Debian:**
```bash
sudo apt update
sudo apt install git
```

### Step 2: Clone the Repository
"Cloning" means downloading a complete copy of the remote repository (the code on the cloud) to your local computer.
```bash
git clone <repository_url>
cd <repository_name>
```

### Step 3: Open in VS Code
We highly recommend using **Visual Studio Code (VS Code)**. Once you open the project in VS Code, look at the left sidebar. You will see a **Source Control icon** (it looks like a branch). This tab will visually show you all your Git details, including which files you have changed!
